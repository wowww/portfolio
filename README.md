## 3. Drawing board

> 작업자: Lia(100%)

- **Vanilla JS**와  **Canvas API**를 이용한 그림판입니다.   
- 하단에서 원하는 색상을 골라 캔버스 안(하얀 배경)에 그림을 그릴 수 있습니다.
- 캔버스 창 아래의 바를 사용하여 **브러쉬 크기를 조정** 가능합니다.
- **FILL button**을 누르고 색상 선택 후, 캔버스를 누르면 전체 배경 색상 변경이 가능합니다.
- **PAINT button**을 누르면 다시 브러쉬로 돌아갈 수 있습니다.
- **SAVE button**을 누르면 사진 저장이 가능합니다. (png로 저장)


![](https://i.imgur.com/O4EeQUU.jpg)  

- [**데모 페이지-클릭시, 페이지로 이동합니다.**](https://wowww.github.io/url/paintJS/index.html)

  ``` 
  작업 기간: July 20, 2020 (1일)
  ```
